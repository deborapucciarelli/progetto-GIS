# Progetto GIS

## Requisiti

Assicurati di avere installato:

- Python 3.x
- Node.js e npm
- Git

## Installazione dell'ambiente Python

1. Creare e attivare l'ambiente virtuale (nella cartella backend):
   ```bash
   python -m venv venv
   venv\Scripts\activate   # Su Windows
   
2. Installare le dipendenze Python
  ```bash
pip install -r requirements.txt
```

2. Avvio del backend

Apri il terminale e spostati nella cartella backend attiva l'ambiente virtuale (se non gi√† attivo):
```bash
venv\Scripts\activate   # Windows
```

Avvia il server backend:
```bash
python percorsi.py
```

3. Avvio del frontend

Apri un altro terminale e spostati nella cartella frontend e avvia il server frontend:

```bash
node server.js
```
